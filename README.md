
# 🎬 Movie & TV Show Review Platform
A full‑stack Flask application that lets users explore movies and TV shows, read metadata, view trending content, and browse entertainment news — all powered by the TMDB API and NewsAPI.
---
## 📌 Features
- 🔍 Search Movies & TV Shows
- ⭐ Trending & Popular Content
- 📰 Entertainment News Integration
- 🗂️ Clean, Modular Architecture
- 🖥️ Responsive UI
- 🧪 Error Handling & Validation
---
## 🏗️ Tech Stack
| Layer | Technology |
|---|---|
| Backend | Flask | 
| APIs | TMDB API, NewsAPI | 
| Frontend | HTML, CSS, JavaScript | 
| Database | PostgreSQL | 
| Deployment | AWS EC2 | 
---

##  🌐 API Usage
### TMDB
- Used for:
  - Movie metadata
  - TV metadata
  - Images
  - Trending & popular lists
  - Search endpoints

### NewsAPI
- Used for:
  - Entertainment news
  - Movie/TV industry headlines
